Failures

I had no clue that to get an html document to load on a browser it needs to be label ".html" otherwise it doesn't recognize it as one. I had a problem with the footer floating into the middle of the page due to an error in the css, which is was particulary hard to figure out what was causing it. I learned a lot from YouTube including that I needed a folder for images with the images in the website folder to get photoes to appear on the website. Figuring out how to turn the htm document into an actual website was challenging as I had no clue how to do that. The hardest problem I had to deal with was getting the hamburger menu to work at all. It turns out I'd forgotten a key part, the javascript.

Conclusion

HTML, CSS, and Javascript are challenging to learn but not as difficult as I'd thought. I also learned that small mistakes can lead to huge errors like not being able to load the document in a browser. Fortunately there are lots of packages that can be added onto Atom Text Editor to make the whole process much easier.
